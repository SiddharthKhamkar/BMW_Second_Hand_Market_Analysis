BMW Second-Hand Market Analysis using Excel

📌 Project Highlights

🔹 Analyzed a BMW Second-Hand Market dataset sourced from Kaggle.

🔹 Performed data cleaning and transformation using Power Query to prepare the dataset for analysis.

🔹 Applied various Excel formulas and functions for data processing and business calculations.

🔹 Utilized advanced lookup functions:

XLOOKUP
VLOOKUP
HLOOKUP
INDEX + MATCH

🔹 Created Pivot Tables to summarize and analyze large volumes of data efficiently.

🔹 Designed Pivot Charts for visual representation of key market trends.

🔹 Developed an Interactive Excel Dashboard to present insights in a user-friendly format.


📊 Dashboard Insights


✔ Average price analysis by BMW model

✔ Year-wise sales and pricing trends

✔ Budget vs Premium vehicle comparison

✔ Model-wise performance analysis

✔ Car running (mileage) insights

✔ Transmission-wise analysis (Automatic vs Manual)

✔ Fuel type and engine size comparison

✔ Overall market performance summary

Excel Formulas & Functions
